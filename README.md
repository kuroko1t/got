# got
got is simple tmux tool

## featurs
- new session
- attach session
- kill session

## Required
tmux

## Instllation/Update
```
$ go get -u github.com/skanehira/got
$ got
```

## Uage
| Operation | Key                            |
|-----------|--------------------------------|
| quit      | <kbd>Tab</kbd>  + <kbd>d</kbd> |
| cancel    | <kbd>Ctrl</kbd> + <kbd>c</kbd> |
| select    | <kbd>Enter</kbd>               |
